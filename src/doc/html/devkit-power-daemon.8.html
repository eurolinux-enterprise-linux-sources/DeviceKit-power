devkit-power-daemon — DeviceKit-power Daemon

Synopsis

devkit-power-daemon [--help]

DESCRIPTION

devkit-power-daemon provides the org.freedesktop.DeviceKit.Power service on the system message bus. Users or administrators should never need to start this daemon as it will be automatically started by dbus-daemon(1) whenever an application calls into the org.freedesktop.DeviceKit.Power service.

OPTIONS

--help

Show help options.

AUTHOR
